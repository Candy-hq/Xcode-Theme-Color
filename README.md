# Xcode-Theme-Color

##自己花了很久配置出来的颜色.
##减少疲劳.


###找到xcode目录下的路径: 前往文件夹 : ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
###将主题导入.重启Xcode即可
